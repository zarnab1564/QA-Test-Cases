## 🌐 About the Project

Certified Mail Labels is an approved USPS online platform for printing certified mail labels with postage and tracking — covering return receipt signatures, delivery confirmation, email notifications, API and automated batch processing for compliance needs. 

## 🧪 Features Covered

### 📬 Label Generation
- USPS Certified Mail label creation & printing
- Label accuracy & format validation
- Address validation & auto-fill
- Envelope type selection

### 📦 Mail Services
- Certified Mail with Electronic Delivery Confirmation
- Certified Mail with Return Receipt Signature
- Priority Mail & Priority Mail Express
- First-Class Mail

### 🔁 Delivery & Tracking
- USPS tracking number generation
- Real-time delivery status tracking
- Email notification flows
- Electronic Proof of Acceptance
- Electronic Return Receipt (Green Card replacement)

### ⚙️ Batch Processing & API
- Bulk label generation
- REST API integration testing
- Automated batch processing flows
- CSV/PDF upload & processing

### 👤 User & Account Flows
- User registration & login
- Return address management
- Mailing history & records
- 10-year compliance archive access

### 💳 Billing & Payments
- Pay-as-you-mail flow
- No hidden fee validation
- Cost calculation accuracy
